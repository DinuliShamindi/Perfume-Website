# LORITO Perfumes Website

A premium, multi-page website for LORITO, a luxury perfume brand.

## Features
- **Modern & elegant design**: Dark mode with gold accents.
- **Responsive**: Fully responsive and mobile-friendly.
- **Animation**: Smooth scroll animations using AOS library.
- **Custom Cursor**: Enhanced user experience with a custom cursor.
- **Pages**:
  - `index.html`: Stunning landing page.
  - `products.html`: Collection grid.
  - `about.html`: Brand story and process.
  - `contact.html`: Contact form.

## Technology Stack
- **HTML5**: Semantic structure.
- **Tailwind CSS (CDN)**: Utility-first CSS framework for rapid styling.
- **Vanilla JavaScript**: Interactions and custom cursor logic.
- **AOS**: Animate On Scroll library.
